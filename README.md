# Socket Mini Project

This project is a simple introduction to how networking works using Python’s socket module.

It takes a hostname and resolves it into an IP address using DNS lookup. This helped me understand how devices actually find each other on the internet instead of just “going to a website”.

## What I learned
I learned how TCP/IP and sockets work together at a basic level, and how browsers rely on these systems to communicate with web servers.

## What surprised me
I didn’t expect something like DNS lookup to be this simple in Python. It’s cool how a few lines of code can interact directly with the internet.

## What I want to learn next
I want to understand how servers handle multiple connections at the same time without crashing or slowing down.
